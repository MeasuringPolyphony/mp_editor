About the Score Editor

The Score Editor is for making simple edits, including to pitches and rhythms. If you made mistakes in your initial entry of the music on the previous screen (the Music Input screen) the score will not align properly and you may need to make adjustments to the note durations or add dots in order to get the proper alignment.

Making Simple Edits

When you click on a pitch, the staff to which that pitch belongs will be highlighted with a dashed box in the manuscript image in the left of the screen. This is to help you check the accuracy of the edits you make.

You can make simple edits in the Score Editor directly as follows. By clicking on a pitch to highlight it you can edit:

  • Pitch: Use up/down arrow keys to change pitch.
  • Rhythm: Type a number key to alter its rhythm (1 semibreve, 2 minim, 4 semiminim, 8 fusa, 6 semifusa, 0 breve, 9 long, 7 maxima). Scoring-up will rerun after a rhythmic value changes.
  • Dots: Typing a period will add or remove the dot from the highlighted pitch. Scoring-up will automatically rerun.
  • Accidentals: To add a flat, press -. To add a sharp, press #. To add a natural, press n. To clear accidentals, press N.
  • Cloning and Deleting Elements: While it is not possible to insert extensive new musical elements at the score editing phase, a simple way of adding elements can be to utilize the clone feature. Elements can be cloned by selecting one and pressing the I key. Elements can be deleted by pressing Backspace.
  • Text: Unfortunately at this time you cannot make any edits to the underlaid text (lyrics) in the Score Editor. Texts can be edited at the Music Input screen.

You can also use the "Return to Music Input" button to return to the Music Input screen at any time to make more significant edits, such as deleting entire staves and re-entering thme, but any changes made in the Score Editor thus far will be lost!

Changing the Appearance of your Scores

  • If you would like to add barlines, you can select to add barlines following the semibreve (prolation), breve (tempus), or long (modus) mensuration. If you intend to create to work with your MEI score file in Verovio to display on a web page or to save as a PDF file, it is recommended that you add barlines so that automatic system breaks can be generated.
  • You may also prefer to “Switch to Modern Clefs” by checking this box.

Moving to Editorial Mode

When making corrections in the Score Editor it is assumed that you are correcting mistakes you made, and both the MEI Parts and MEI Score files will be updated with these changes. However, once you move to Editorial Mode, the MEI Parts file will no longer be updated since this is a record of the music exactly as it was recorded in the original manuscript. Any further corrections made in Editorial Mode will be encoded in the MEI Score file as editorial corrections. For example, when a dot is added in Editorial Mode: 


    <choice>
        <corr>
            <note dur="semibrevis" oct="3" pname="a" dur.quality="minor"/>
            <dot form="div"/>
        </corr>
        <sic>
            <note dur="semibrevis" oct="3" pname="a"/>
        </sic>
    </choice>

The purpose of the Score Editor is to do enough editing to enable the correct alignment of the score and edit while viewing the manuscript. For more fine-grained editing, it is recommended to download the MEI file and open it in a tool that allows direct editing of the MEI file such as the Verovio Humdrum Viewer or the Verovio Editor.

Once you click the button “Continue in Editorial Mode” you will receive a prompt that the parts-based MEI file will be downloaded locally.

Saving your Work

To download MEI files of your work, click the “Download MEI File(s)” button. This action will download two files:

  • a part-based MEI file that you can use to open in the MP Editor at a later stage to continue your music input or editing work
  • A score-based MEI that you can display on a web page or to save as a PDF file, using Verovio for example.

You can also simply copy the coding of the score-based MEI from the score editor screen by clicking the “Copy MEI Score to Clipboard” button.
