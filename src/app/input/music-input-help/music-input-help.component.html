Music Input Instructions

  • Adding clefs (Video clip)
    • Type C to insert C3 clef, or F to insert F4 clef.
    • Press the up or down arrows to move the clef to another line.
    • You can remove the clef by clicking the backspace/delete key.
  • Adding key signatures (Video clip)
    • Type k to enter key signature mode after adding a clef.
    • Then enter the pitch name letter and accidental (either the keystroke - or #, see below) to add to key signature.
    • To remove from a key signature, enter k, the same pitch letter, and accidental again.
    • Example: For F-sharp, type k f #
    • You can also remove the key signature by clicking the backspace/delete key.
  • Adding pitches (Video clip)
    • Type pitch letter name of each note to add them to the line: f.ed.efgagfgedderfag
    • Type "." to add dot after note.
    • Type "r" to add a rest.
    • Type backspace to delete a note or notes.
    • Use capital letters to choose the larger interval for the next pitch. So if you are on `d` and want to go to the `a` a fifth above, type capital A rather than lowercase (which would give the `a` that is a fourth below).
    • Up/down arrows transpose last pitch up/down a step.
    • Shift-up/down arrows transpose last pitch up/down an octave.
  • Adding Accidentals (Video clip)
    After adding a note:
    • press - to add a flat.
    • press # to add a sharp.
    • press n to add a natural.
    • press N to clear the accidental for that note.
  • Adding ligatures (Video clip)
    Note that while duration of pitches in ligatures can be changed after the fact (in the score editor screen), ligatures must be specified as pitches are added in the music input phase.
    • After adding a note, press [ to start a ligature.
    • After adding a note, press ] to end a ligature.
  • Adding rhythm (Video clip)
    • 1 semibreve, 2 minim, 4 semiminim, 8 fusa, 6 semifusa, 0 breve, 9 long, 7 maxima.
    • Interleaved method: type (new) rhythm before adding a note. The default rhythm is whole note (semibreve).
    • Second-pass method: type pitches first, then select first note and type rhythms to update note rhythms.
    • Selective second-pass method: type clef, then most common rhythm, then notes, then click on notes that need rhythms to fix. Typing rhythm number will change rhythm and advance to next note.
    • left/right arrow keys or tab/shift-tab moves to next/previous note/rest/clef.
  • Adding text (Video clip)
    • Click on a note (or left/right arrows or tab/shift-tab to move to target note).
    • Type text for note, then press the pace, tab, or right arrow key or click to go to next note and enter the next word/syllable.
    • backspace delete last letter.
    • shift-backspace deletes the entire syllable.
    • Adding a dash at the end of text input for a note marks that this is part of a word that continues to the next note, and a hypphen will be seen on the screen.
    • To correct mistyped text, click the note to highlight it and press backspace. Then type the new text that belongs to that note.
  • Repeating tenor (Video clip)
    • When a staff that belongs to the tenor part is selected, the number of statements can be indicated.
    • If more than one tenor statement is indicated, a repeating tenor will be generated.
    • Where the tenor repeats from can be set after entering that element by pressing the "Set End of Repeating Tenor" button. The ending will be highlighted in blue.
    • If no end point is specified, the entire tenor will repeat by default.
