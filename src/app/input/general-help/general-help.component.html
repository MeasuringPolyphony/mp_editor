Manuscript Navigation

  • To select the page of the manuscript you want to work on, either use the scrollbar to the right of the manuscript or type the folio number in the box and click the Go button.
  • Click the magnifying boxes for the appropriate zoom level and move around the manuscript using your mouse or the trackpad.

Metadata Entry

First enter basic metadata about the composition. This includes a short title for the work, composer name (if known), the notation type (selected from a drop-down menu), the siglum for the manuscript (if known). You should add your name by clicking the “Add Contributor” button and selecting the appropriate role (Encoder, Proofreader, Editor) along with your name. You may add multiple contributors, and you may also add your name additional times with alternate roles (for example, if you are both encoder and editor).
If there is something you don't know, you can leave it as is and continue.
Metadata must be entered before staff input can fully begin.

Adding Staves (Video clip)

  • To select an area for a new staff, hold SHIFT and click-and-drag an area surrounding the staff on the Diva.js viewer. When you release the mouse, a staff is added.
  • To select a staff, click on the bounding box on the viewer: it will appear as a solid box when you select it, and unselected staves will have bounding boxes with dashed lines.
  • Bounding boxes can overlap.
  • When a staff is selected, the voice it belongs to and mensuration of that voice can be set on the right. This information is saved automatically. It is important to set the correct voice part and mensuration for each stave so that the scoring-up process (next step) will work correctly. However, you can go back at any point and fix/update either the voice part or mensuration for any stave.
  • You can select as many staves as you need in this process, for all the voice parts. You can do this in any order, and can continue to the music input phase, and then later add more staves.
  • To add musical components (pitches, rhythm, text), begin typing. This will add musical elements to the staff below the voice and mensuration options. Information on this process is available under the "Music Input Help" Tab.
  • To delete a staff, first select it and then click the "Delete Staff" button under the "Staff Input" tab.

Saving Work on the Music Input Screen

The “Download MEI File” downloads any music input at this stage of the process as a parts-based MEI file. You can reload this parts-based MEI file at a later point from the home screen of the MP Editor (using the File chooser under the heading “Load an .mei file").

You also have the option of downloading the Humdrum data by clicking on the button “Download System Humdrum.”
